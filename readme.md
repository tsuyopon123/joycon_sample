# joycon-sample

Nintendo Switch Joy-Con demo

## これはなに?
Nintendo Switch Joy-Con の姿勢をリアルタイム描画するアプリケーションのデモ

## 環境
- Nintendo Switch Joy-Con (L)
- Windows10 21H2
- Python3.10.1

## 導入
```shell
pip install -r requirements.txt
```

## 使い方
1. `main.py` を実行
    ```shell
    ./main.py
    ```
1. `joycon_rotation.pde` を実行
